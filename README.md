# 📊 Result Management System

A simple Python CLI application for managing student results. The system calculates grades, determines pass/fail status, and stores records in a JSON file for future access.
<br>
This project is beginner-friendly and demonstrates the use of Python fundamentals such as functions, loops, conditionals, exception handling, file handling, and JSON data storage.

## ✨ Features

* Add student results
* Calculate total marks and percentage
* Assign grades automatically
* Determine pass/fail status
* Store data in `results.json`
* Update existing student records
* View individual student results
* View all class results
* Input validation and error handling

## 🛠 Tools & Technologies

* Python 3
* JSON
* File Handling
* Exception Handling
* Command Line Interface (CLI)

## 🚀 How to Run

```bash
git clone https://github.com/rachitries/result-management-system.git
cd result-management-system
python result_management_system.py
```
or download result_management_system.exe
